QUANTUMCOACH APP SETUP (Streamlit Cloud)

1. Go to https://streamlit.io and create an account (or log in).
2. Click "New app" and connect a GitHub repo, or use their file uploader tool.
3. Alternatively, deploy locally using the following steps:

Local Setup (Optional):
- Install Python (https://www.python.org/downloads/)
- Open Terminal or Command Prompt
- Run:
  pip install streamlit openai pandas
  streamlit run app.py

This will launch the app in your default web browser.

If uploading to Streamlit Cloud:
- Upload all files in this ZIP to a GitHub repo or use their app builder
- Confirm public visibility to share the app link with others
